## For developers

### Setup

Open this repository in devcontainer.

#### Install dependencies

```bash
npm install
```

#### Register your RESAS API key

```bash
cp .env.example .env

# edit .env file
code .env
```

### Start Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
